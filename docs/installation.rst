Installation
=====

pyClim, first install it using pip:

.. code-block:: pip install pyclim

For testing purposes, you can also install from TestPyPI:

.. code-block:: pip install -i https://test.pypi.org/simple/ pyclim_test



System requirements
-------------------
- Python 3.10 or above.

- The package is compatible with major operating systems (Linux, macOS, Windows).
